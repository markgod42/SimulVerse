Choose Your Wallet Interface

Mark supports multiple wallet interfaces. Choose the one that doesn't make you want to debug reality.

🦊 MetaMask
Latest & Greatest (2025 Edition)
✅ Pros
  • Manage Ethereum + Solana with one wallet
  • Native Solana support (finally!)
  • Familiar interface (no learning curve)
📋 Requirements
  • MetaMask Extension v11.16.0+
  • Minimum 0.005 SOL
1 Check MetaMask Update
  1. Open MetaMask extension in Chrome/Firefox
  2. Click top-right menu (⋯) → "Settings"
  3. Check version in "About" tab
  4. Ensure it's v11.16.0 or higher
2 Create Solana Account
  1. On MetaMask main screen, click account dropdown
  2. Click "Add account or hardware wallet"
  3. Select "+ Solana account" option
  4. Enter account name (e.g., "Mark's Rewards")
  5. Click "Create"
3 Deposit SOL
  1. Select your new Solana account
  2. Click "Receive" button
  3. Copy wallet address
  4. Send at least 0.005 SOL from exchange
4 Add $MARK Token (Create ATA)
  1. In Solana account, click "Tokens" tab
  2. Click "Import tokens" or "+" button
  3. Enter Token Address: 4200000000000000000
  4. Click "Add" and confirm transaction

Debug Common Issues

❌ "Insufficient funds" error
Solution:

Deposit at least 0.005 SOL into your wallet. Mark's simulation requires proper funding.

❌ Token not visible in wallet
Solution:
  • Refresh wallet (F5 or restart app)
  • Retry manual token addition
  • Check internet connection (Mark's servers are sometimes slow)
❌ Transaction failed
Solution:
  • Retry after 5-10 minutes
  • Avoid network congestion times
  • Increase transaction fee (advanced settings)
❌ Incorrect Token Address
Verification Method:
  • Correct format: 44-character Base58 string
  • Example: 99zqUzQGohamfYxyo8ykTEbi91iom3CLmwCA75FK5zTg
  • Incorrect: Addresses starting with 0x (that's Ethereum, not Solana)
